How is a melody generated from numbers?

The goal is to create a melody from the prime decomposition of the number.

To make it short, the melody lasts n beats (between 1 and 32), so the given number is decomposed in n numbers of equals length. Each number is decomposed in prime factors. Each cell represents a prime number. The row on which the prime factor is located determines which note will be played (from D5 to D6). The melody will is generated by playing one beat (column) after the other.
